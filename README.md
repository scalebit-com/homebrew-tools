# Scalebit-com Tools

Homebrew tap for tools by [Scalebit](https://github.com/scalebit-com).

## cabidl

Component Architecture Boundary and Interface Definition Language (CABIDL) — parse and validate architecture specification files.

### Install

```bash
brew install scalebit-com/tools/cabidl
```

Or tap first, then install:

```bash
brew tap scalebit-com/tools
brew install cabidl
```

### Install a specific version

```bash
brew install scalebit-com/tools/cabidl@0.1
```

### Verify

```bash
cabidl --version
```
